
No Git, **HEAD** é um ponteiro que indica **onde você está agora** no histórico do repositório. **HEAD** é o último commit, commit mais recente 
![[Pasted image 20260623102617.png]]


```
git show {hash do commit}
```


```
git diff
```
mostra **as diferenças entre o que você alterou nos arquivos e o que ainda não foi preparado para commit**.

```
git diff {commit-mais-antigo}..{commit-mais-novo}
```
