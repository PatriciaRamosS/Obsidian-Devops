`git reset` é um comando usado para **voltar o Git para um estado anterior**.

Ele pode desfazer:

- um `git add`;
- um commit;
- alterações locais;
- o ponteiro da branch atual.

A ideia principal é:
```
git reset <para-onde-voltar>
```
## O que colocar em `<para-onde-voltar>`?

Você substitui `<para-onde-voltar>` por um commit, por exemplo:

Significa: “volte 1 commit antes do commit atual”.
```
git reset HEAD~1
```


```
git reset --hard <para-onde-voltar>
```

significa:

**voltar o repositório para um ponto específico do histórico e apagar tudo que foi feito depois desse ponto.**

### Voltar 1 commit

```
git reset --hard HEAD~1
```

Significa: volte para o commit anterior ao atual.

### Voltar 2 commits

```
git reset --hard HEAD~2
```

Significa: volte dois commits antes.
