
O README é um arquivo com extensão .md, escrito em Markdown, uma linguagem de marcação que converte texto em [HTML](https://www.alura.com.br/artigos/html) válido. Caso queira saber mais, consulte este artigo sobre como funciona o [Markdown](https://www.alura.com.br/artigos/como-trabalhar-com-markdown) e como escrever anotações com essa linguagem. 

O arquivo README.md é o cartão de visitas do seu repositório no GitHub. Ele serve para apresentar as informações essenciais do projeto, facilitar a colaboração e informar rapidamente visitantes e recrutadores sobre as funcionalidades, instalação e uso. 

Nas plataformas de repositórios remotos, como o [GitHub](https://docs.github.com/pt/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-readmes), a função desse arquivo é apresentar informações do projeto, como: 

- Descrição do seu projeto; 
- Funcionalidades; 
- Como os usuários podem utilizá-lo; 
- Onde os usuários podem encontrar ajuda sobre seu projeto; 
- Autores do projeto.
