Em uma [atividade do curso anterior](https://cursos.alura.com.br/course/logica-programacao-funcoes-listas/task/173873), vimos que estavam acontecendo alguns erros com a biblioteca `Responsive Voice`. Na atividade, foi fornecido um código alternativo para ser utilizado no lugar da biblioteca. Assim, você pode criar um gist com esse código alternativo.

Assim, seu Gist terá o seguinte código no arquivo `app.js`:

Copiar

```javascript
if ('speechSynthesis' in window) {
        let utterance = new SpeechSynthesisUtterance(texto);
        utterance.lang = 'pt-BR';
        utterance.rate = 1.2;
        window.speechSynthesis.speak(utterance);
} else {
    console.log("Web Speech API não suportada neste navegador.");
}

```

Repare que teremos apenas o arquivo `app.js` no Gist, uma vez que, para usar a API Web Speech não precisar importar nenhum script no arquivo HTML.

Agora você pode compartilhar o novo código!
