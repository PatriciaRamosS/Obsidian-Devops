### 1. Ícone de arquivos — **Explorer**

É o primeiro ícone, parecido com duas folhas.

Serve para mostrar os arquivos e pastas do projeto.

Na sua imagem ele está aberto e mostra:

```
NUMERO-SECRETO
├── img
├── app.js
├── index.html
└── style.css
```

Você usa para abrir, criar, renomear e apagar arquivos do projeto.
![[Pasted image 20260617110137.png]]

-------
### 2. Lupa — **Search / Pesquisa**

É o ícone de lupa.

Serve para pesquisar palavras, códigos ou textos dentro dos arquivos do projeto.

Exemplo: procurar onde aparece a palavra:

```
tentativas
```

![[Pasted image 20260617110102.png]]

-------
### 3. Ramificação — **Source Control / Controle de Código**

É o ícone parecido com uma linha se dividindo.

Serve para usar Git dentro do VS Code.

Você usa para ver arquivos modificados, fazer commit, controlar versões e acompanhar alterações antes de enviar para o GitHub.

![[Pasted image 20260617110044.png]]

---

### 4. Play com inseto — **Run and Debug / Executar e Depurar**

É o ícone de “play” com um bichinho.

Serve para executar e depurar o código.

Depurar significa rodar o programa passo a passo para encontrar erros.
![[Pasted image 20260617110205.png]]

---

### 5. Quadradinhos — **Extensions / Extensões**

É o ícone com quatro blocos/quadrados.

Serve para instalar extensões no VS Code.

Exemplos:

```
Live Server
Prettier
Python
GitHub Pull Requests
JavaScript Debugger
```
![[Pasted image 20260617110215.png]]
