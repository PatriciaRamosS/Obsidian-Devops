-> GitHub: é uma plataforma online onde podemos armazenar repositórios Git, compartilhar código e colaborar com outras pessoas em projetos.

-> Git: é uma ferramenta de controle de versão. Ele registra o histórico de alterações feitas nos arquivos de um projeto, permitindo acompanhar mudanças, voltar para versões anteriores e trabalhar em diferentes versões do código.

**Git = controla as versões do código na sua máquina.**  
**GitHub = guarda e compartilha esse código na internet.**

Exemplo:

Você cria um projeto no computador e usa o **Git** para salvar o histórico das alterações. Depois, envia esse projeto para o **GitHub** para deixar salvo online e permitir que outras pessoas vejam ou colaborem.
